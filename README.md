# Blue-Player

A modern static music player UI built with HTML, CSS and JavaScript.

## Overview

This project is a lightweight music player interface featuring:
- Responsive navigation and section switching
- Playlist slider for trending and featured songs
- Audio controls: play/pause, next, previous, shuffle, repeat
- Volume control and progress tracking
- Search and section-based song browsing
- Local storage support for persisting songs added during the session

## Built With

- HTML
- CSS
- JavaScript
- Font Awesome (CDN)
- Swiper.js (CDN)

## Project Structure

- `index.html` - main application markup
- `css/style.css` - app styling and responsive layout
- `js/script.js` - player behavior and UI handling
- `assets/` - images, audio files, and media resources

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in a web browser.
3. Use the player controls to browse and play songs.

## Notes

- The app runs entirely in the browser and does not require a server.
- External icons and slider styles are loaded via CDN links in `index.html`.
- Audio file paths are referenced from the `assets/music` folder.
